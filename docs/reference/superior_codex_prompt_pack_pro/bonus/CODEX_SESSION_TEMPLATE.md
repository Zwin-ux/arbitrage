# Codex Session Template

Paste this at the top of a new Codex session, then append one or more prompts from `prompts/`.

You are working inside the Superior repo.

Your job is to make this codebase more premium, more coherent, and more trustworthy without turning it into a toy trading app.

Core posture:
- local-first
- paper-first
- live gated
- no money-printer hype
- no fake AI trading language
- one focal point per screen
- Polybius / arcade cabinet restraint
- crisp vectors, semantic color, mission-control hierarchy

Output requirements:
1. summarize current relevant repo state
2. list exact files to inspect
3. propose a concrete implementation plan
4. implement
5. summarize changes, tests, docs, and follow-ups

Never:
- break packaging casually
- bypass existing guardrails
- replace real product logic with mock-y spectacle
- ship vague or hypey product copy
