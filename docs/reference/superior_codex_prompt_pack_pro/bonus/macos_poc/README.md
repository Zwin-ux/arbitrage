# macOS PoC Bonus Templates

These files are starter assets for Codex or a human maintainer. They are not a finished shipping lane.

Included:
- `build-macos-release.sh`: starter build script
- `Info.plist`: starter app metadata
- `entitlements.plist`: starter entitlement scaffold
- `com.superior.recorder.plist`: LaunchAgent starter template
- `release-macos.yml`: starter GitHub Actions workflow

Use these with:
- `prompts/20_macos_poc_cross_platform_qt.md`
- `prompts/21_macos_packaging_and_launchagent.md`
- `prompts/22_ci_matrix_and_release_pipeline.md`
