# SUP v1 Desktop

React/TypeScript rehearsal machine scaffold for the consumer v1 loop.

## Current state
- deterministic local tapes
- replay and tutorial shell
- hold-to-commit interaction
- bundled Tauri shell scaffold

## Bring-up
- `npm install`
- `npm run check`
- `npm run test`
- `npm run build`

## Native shell note
`tauri:dev` and `tauri:build` require a Rust toolchain on the machine.
